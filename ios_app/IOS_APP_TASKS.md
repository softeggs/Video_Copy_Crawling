# iOS App Tasks

## 当前目标

- [x] 默认入口保持统一后端模式，不回退到飞书直连
- [x] 默认后端地址使用 `http://192.168.0.32:8002`
- [x] 登录、提交、列表、详情、概览、删除、收藏、处理中状态、快捷指令密钥能力已接入统一后端
- [x] 启动恢复增加严格会话校验：本地有 token 时先调用 `GET /auth/me`
- [x] `401/403` 会话失效时立即清空本地会话并回到登录页
- [x] 启动阶段增加“恢复中”状态，避免先进入主界面再闪回登录页

## 本轮完成

### 认证与会话

- [x] `AuthServiceProtocol` 增加 `fetchCurrentUser(token:)`
- [x] `APIService` 增加 `GET /auth/me`
- [x] `AuthManager.restoreSession()` 改为“本地读取 + 远端校验”
- [x] 校验成功后以服务端返回的 `UserDTO` 覆盖本地缓存
- [x] 校验失败后立即执行登出并清理持久化数据

### 页面与行为

- [x] `ContentView` 根据 `isRestoringSession` 显示恢复中界面
- [x] 保持 `AppConfig.useFeishuDirect = false`
- [x] 保持 `AppServices` 默认绑定 `BackendAuthService` 与 `BackendVideoRepository`
- [x] 统一 `401` 失效通知，避免鉴权过期后长期停留在假登录态

### 测试

- [x] 增加 `/auth/me` 请求测试
- [x] 增加本地会话恢复成功测试
- [x] 增加会话恢复失败后自动清理测试
- [x] 保留原有登录、DTO 解码、错误映射测试
