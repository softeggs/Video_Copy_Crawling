# 定时任务快速开始

## 5 分钟快速上手

### 1. 确认配置 ✅

检查 `.env` 文件中的飞书配置：

```env
FEISHU_APP_ID=cli_xxx
FEISHU_APP_SECRET=xxx
FEISHU_BITABLE_APP_TOKEN=xxx
FEISHU_BITABLE_TABLE_ID=xxx
```

### 2. 测试一次 🧪

手动执行一次检查：

```bash
python check_blank_links.py
```

如果看到类似输出，说明配置正确：

```
开始检查飞书表格中的空白链接...
共发现 X 条空白记录
处理进度: 1/X
...
处理结果:
  总计: X 条
  成功: X 条
  失败: 0 条
```

### 3. 启动定时任务 🚀

#### Windows 用户

双击 `run_scheduler.bat` 或执行：

```bash
python scheduler_app.py
```

#### Linux/Mac 用户

```bash
python scheduler_app.py
```

### 4. 验证运行 ✔️

看到以下输出说明运行正常：

```
============================================================
飞书表格定时任务启动
检查间隔: 300秒 (5分钟)
按 Ctrl+C 停止
============================================================
```

### 5. 添加测试链接 📝

在飞书表格中添加一条测试记录：

| 原始链接 | 标题 | 作者 | 处理状态 |
|---------|------|------|---------|
| https://www.bilibili.com/video/BV1xx... | (空) | (空) | (空) |

等待 5 分钟，定时任务会自动处理这条记录。

### 6. 查看结果 👀

刷新飞书表格，应该看到：
- ✅ 标题已填充
- ✅ 作者已填充
- ✅ 一句话总结已生成
- ✅ 处理状态变为"已完成"

## 常用命令

```bash
# 手动检查一次
python check_blank_links.py

# 启动定时任务（持续运行）
python scheduler_app.py

# 启动 Web 界面
streamlit run app.py
```

## 停止定时任务

按 `Ctrl+C` 即可停止。
