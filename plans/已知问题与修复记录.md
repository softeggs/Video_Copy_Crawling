### BUG-002：日志文件无限增长

| 字段 | 内容 |
|------|------|
| 严重程度 | 中 |
| 状态 | ⚠️ 未修复 |
| 发现时间 | 2026-03-12 |
| 影响范围 | `logs/app.log`（已达 490 万行） |

**现象：**  
`logs/app.log` 使用简单的 `FileHandler`，无大小限制，随定时任务持续运行无限增长，存在磁盘耗尽风险。

**修复方案：**  
将 `utils/logger.py` 中的 `FileHandler` 替换为 `RotatingFileHandler`：
```python
from logging.handlers import RotatingFileHandler

handler = RotatingFileHandler(
    log_file,
    maxBytes=50 * 1024 * 1024,  # 50MB
    backupCount=5,
    encoding='utf-8'
)
```

**相关文件：** `utils/logger.py`
